# 产品调整会审工具

这是由原“南京德基新店开业总控驾驶舱”替换升级而来的产品调整协作工具。

## 线上架构

- GitHub Pages：提供公网页面
- 阿里云函数计算 + OSS：保存团队共享数据
- GitHub Actions：`main` 分支更新后自动发布

## 当前能力

- 热厨、水吧、烘焙分别维护多道上架、下架与调优事项
- 事项类别、菜品名称、执行时间、会审部门、执行说明可编辑
- 研发、市场、营运、厨政、采购会审意见可编辑
- 通过统一入口新增、切换与删除产品调整事项
- 云端自动保存与每 15 秒同步
- 一键导出 PDF
- 品牌 VI 蓝色与橙色视觉

## 云端数据

工具的完整共享状态保存在阿里云 OSS Bucket 的专用文件中：

```text
/dashboard-state.json
```

函数计算只能读写该 Bucket，网页不包含阿里云账号密钥。

## 发布

推送到 `main` 分支后，`.github/workflows/pages.yml` 会自动部署 GitHub Pages。
